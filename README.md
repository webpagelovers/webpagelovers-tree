# Description
```
This is a Vue component for selecting tree data nodes
```
# Install

``` 
npm install webpagelovers-tree
```
# Use

```
# In main.js
import WebpageloversTree from 'webpagelovers-tree'
Vue.use(WebpageloversTree)

# In *.vue
<webpagelovers-tree></webpagelovers-tree>  
```
